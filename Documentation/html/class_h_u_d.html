Detailed Description

Classe représentant l'interface utilisateur tête haute (HUD) (Head Up Display en anglais).

Le HUD est utilisé pour afficher des informations à l'écran, telles que la position d'un objet, un texte, etc. Ici, il est utilisé pour afficher les informations du joueur au cours du jeu, sa barre de vie et sa tête et son nom
